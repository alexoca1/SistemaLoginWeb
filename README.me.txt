# Proyecto de Login

Este es un proyecto de página de login desarrollado con HTML, CSS y PHP. El proyecto incluye un formulario de inicio de sesión y un sistema básico de validación de usuarios.

## Características

- Formulario de inicio de sesión con validación.
- Diseño responsivo que funciona en diferentes tamaños de pantalla.
- Mensajes de error estilizados para una mejor experiencia de usuario.
- Uso de un sistema de grid para organizar el contenido.

## Instrucciones de Uso

1. Clona este repositorio en tu máquina local.
2. Asegúrate de tener un servidor web (por ejemplo, XAMPP, WAMP) configurado para ejecutar archivos PHP.
3. Coloca la carpeta del proyecto en la raíz de tu servidor web.
4. Abre `index.html` en tu navegador web para acceder al formulario de login.
